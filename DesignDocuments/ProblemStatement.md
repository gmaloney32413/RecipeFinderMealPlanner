Project Description:
This project will be a Java-based web application that allows users to search for recipes using an external recipe API and save them to a personal meal planner. Users will be able to browse recipes, view detailed information such as ingredients and instructions, and organize selected recipes into a weekly meal plan.

The application will integrate with a public recipe API to retrieve recipe data. Users will be able to add recipes to their personal collection, update notes or meal plan assignments, and remove recipes they no longer want saved.

Project Goals:
The goal of this project is to demonstrate the development of a Java web application that integrates external APIs, performs CRUD operations with a database, and provides an interactive user interface for managing personal data.

Key Features:

Search recipes using an external recipe API
View recipe details including ingredients and instructions
Save recipes to a personal favorites list
Add recipes to a weekly meal planner
Add notes to saved recipes
Remove recipes from favorites or meal plans

Technology Stack:

Java (Servlets/JSP)
REST API integration for recipe data
MySQL database for storing saved recipes and meal plans
HTML, CSS, and Bootstrap for the user interface
GitHub for version control

Version Control Plan:
The project will be maintained in a GitHub repository. Development will be tracked through commits and branches for new features. GitHub issues will be used to track progress and feature implementation.

Testing Plan:
Testing will include unit testing for DAO classes and service classes that interact with the database. Additional testing will include verifying API responses, ensuring CRUD operations function correctly, and performing manual testing of the user interface to confirm user workflows operate as expected.

Future Enhancements (Optional):

User login and authentication
Automatic grocery list generation based on meal plans
Nutritional information display
Recipe ratings and reviews
